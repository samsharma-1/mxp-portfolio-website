## **MXP Portfolio Website** 

## **Overview** 

MXP Portfolio is a modern esports strategy and performance portfolio platform designed to showcase the expertise, achievements, and services offered by MXP. 

The website highlights coaching, analytics, tactical planning, performance improvement support, and professional esports development through a clean and responsive interface.
